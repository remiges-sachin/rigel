# rigelctl

## Build

```
$ go build -o rigelctl main.go
```

## Run

```
./etcdctl --etcd-endpoint localhost:2379,localhost:2380,localhost:2390 --app starmf --module ucc schema add tmp/sample_schema.json
```

## set a config key
```
./etcdctl --etcd-endpoint localhost:2379,localhost:2380,localhost:2390 --app starmf --module ucc schema add tmp/sample_schema.json
```
